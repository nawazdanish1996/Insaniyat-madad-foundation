# Insaniyat Madad Foundation Website

A responsive, modern, and inclusive landing page for the **Insaniyat Madad Foundation**, a social welfare initiative based in Goalpokher, Uttar Dinajpur. The website is designed to showcase the organization's mission of serving humanity regardless of religion, caste, or creed.
